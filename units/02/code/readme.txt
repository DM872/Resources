Exercise session 24.02.2026

1. Work with the main_ATSP_template.py to model the asymmetric TSP
2. Implement D-L or G-G or G-G m. formulation of the ATSP
3. Implement the separation routine of DFJ inequalities at the root node of the implemented ATSP formulation
4. Separate and impose all violated DFJ inequalities at the root node of the MILP formulation and note the improved performance of the formulation
5. Implement the fractional callback where separation routine is called to generate violated DFJ cuts during the branch-and-bound process

In the end, you should obtain codes for a cut-and-branch approach for the ATSP (#4) and a branch-and-cut approach for the ATSP (#5) and be able to compare performaces of the two.
